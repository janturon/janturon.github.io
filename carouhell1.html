Back to Carouhell

To create a carousell from list, add a class carouhell to it. The carousel below should rotate from right to left with a period of 1000ms specified in the data-stay attribute. Set data-hover-pause to pause on mouse over. The client code sets the carousel size (400x400) and the transition duration (400ms) in the style. You can add a third image anytime you click the button which calls the client function addImage().

To make this stuff working, just add the carouhell.css (333 B) and the carouhell.js (666 B). No other dependencies, see the code snippet below (which has also 666 B).

<link rel="stylesheet" type="text/css" href="src/carouhell.css"> <script src="src/carouhell.js"></script> <style> .carouhell { width: 400px; height: 400px; background: gray; } .carouhell li { transition: 400ms; } </style> <script> function addImage(src) { var li = document.createElement("li"); li.innerHTML = `<img src="${src}">`; document.getElementById("test").appendChild(li); } </script> <button onclick="addImage('assets/img3.jpg');this.disabled=1">Add the third image</button> <br><br> <ul id="test" class="carouhell" data-stay="1000"> <li><img src="assets/img1.jpg"> <li><a href="#"><img src="assets/img2.jpg"></a> </ul>
