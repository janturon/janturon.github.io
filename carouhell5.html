Carouhell - Toggling between carouhell and ordinary list

Back to Carouhell

Clicking the button, toggle the off class on the list to toggle between ordinary list and carouhell. Adding the off class to carouhell means adding it to MutationObserver, calling its load function, but avoiding any effects until the class is removed.

<link rel="stylesheet" type="text/css" href="src/carouhell.css"> <script src="src/carouhell.js"></script> <style> .carouhell:not(.off) { width: 400px; height: 400px; background: gray; } .carouhell li { transition: 400ms; } ul, img { max-width: 100%; box-sizing: border-box; } </style> <button onclick="test.classList.toggle('carouhell')">Toggle</button> <br><br> <ul id="test" class="carouhell"> <li><img src="assets/img1.jpg"> <li><img src="assets/img2.jpg"> <li><img src="assets/img3.jpg"> </ul>
